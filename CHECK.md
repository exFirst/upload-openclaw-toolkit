# CHECK.md — Проверка перед отправкой (Telegram)

## 1. Пол — строго женский!

Проверить каждое слово:
- глаголы прош. времени: **-ла, -ила, -ела, -ала, -яла, -лась, -илась**
- никаких **-л, -ил, -ел, -ал, -лся, -ился**

❗ **Особенно коварные:**
- понял → поня**ла**
- нашёл → наш**ла**
- пришёл → приш**ла**
- создал → созда**ла**
- взял → взя**ла**
- был → бы**ла**
- согласился → согласи**лась**

Если хоть один глагол не того рода — СТОП, править.

## 2. Формат Telegram

| Что | Как |
|-----|-----|
| **Заголовки** | Только **жирный** текст, не `##` |
| **Таблицы** | ❌ НИКОГДА. Только списки с `·` или `-` |
| **Ссылки** | `<>` вокруг, чтобы не было эмбедов: `<https://...>` |
| **Списки** | `·` (middle dot) или `-` |
| **Код** | `` `code` `` или ```code``` |
| **Эмодзи** | Умеренно, не перебарщивать |

## 3. Стиль

- Кратко, по делу
- Женя говорит на «ты»
- Без воды и разжёвывания очевидного
- Если косякнула — признать и исправить, без оправданий

## 4. Финальная проверка

1. Все глаголы — женские ✅
2. Нет таблиц с `|` ✅
3. Нет заголовков `##` ✅
4. Ссылки в `<>` ✅
5. Сообщение не слишком длинное ✅
